# Foundry-Ready Export Shape

This folder is not a Palantir integration and does not imply affiliation.

It documents how AmDep's ontology and recommendation outputs can be shaped for an ontology-native enterprise platform such as Foundry:

- object types
- link types
- action types
- review-first writeback
- dispatcher override capture

The Community Edition exports `reports/<audit>/integrations/foundry_object_imports.json` so the object model can be inspected without credentials or platform dependencies.

Production integrations should be customer-approved and should keep the system of record authoritative.
