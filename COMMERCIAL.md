# AmDep Editions

## Community Edition

The public repository is the shareable proof package:

- MIT license.
- Synthetic demo data.
- Reference OR-Tools dispatch optimizer.
- Synthetic Brain Layer models.
- CSV import/export contracts.
- Static audit packet generation.
- Adapter-ready recommendation files for common contractor software stacks.

This is enough for a contractor, operator, investor, or technical partner to run the demo and understand the deployment intelligence problem.

## Commercial Edition

The paid product should stay private:

- customer-calibrated optimizer policies
- customer-trained predictive models
- production API connectors
- secure customer data pipelines
- dispatcher override learning
- private benchmarking
- deployment-specific tuning
- implementation support and audit delivery
- hosted command center, if needed

The algorithm is the product. The public repo should prove the architecture. The commercial layer should carry the operational edge.
